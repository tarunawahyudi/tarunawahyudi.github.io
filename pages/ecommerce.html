E-commerce Digital Printing

Description

ini adalah website ecommerce yang digunakan untuk penjualan perangkat digital pinting PT. DWI SASTRO PRATAMA. Dengan adanya website ini, user dapat mencari produk yang sepesifik sesuai kebutuhan dengan harga yang disesuaikan. Website ini dibuat menggunakan PHP Framework yaitu CI 3 dan database MySQL. Terdapat panel admin yang bisa digunakan untuk manajemen produk dan juga master data lainnya.

Technology

  • PHP
  • Codeigniter 3
  • MySQL
