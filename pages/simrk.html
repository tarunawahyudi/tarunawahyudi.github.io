Sistem Informasi Manajemen Risiko

Description

Sistem Informasi Manajemen Risiko atau disingkat SIMRK adalah proyek BPJS Ketenagakerjaan yang akan di implementasikan di setiap kantor seluruh Indonesia.

Saya berkontribusi penuh dalam pembuatan front-end dan sebagian untuk backend nya. Proyek ini menggunakan teknologi Low Code untuk front-end nya dan Nodejs untuk backend nya. Alasan menggunakan teknologi tersebut karena singkat nya waktu yang diberikan oleh client.

Technology

  • Low Code
  • Wavemaker
  • Node JS
  • Oracle Database
