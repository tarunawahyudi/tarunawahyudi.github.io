Code Resto - Web App

Description

Code Resto adalah sebuah project untuk menyelesaikan submission dari kelas Front end developer expert. Untuk fitur nya sederhana, user bisa mencari restoran dan menambahkan nya kedalam list favoite.

Namun yang menjadi kelebihan dari website ini adalah, website ini menggunakan best practice dari ilmu front end. Pembuatan nya menggunakan web component, mvp pattern, responsive mobile first, optimasi performa, penggunaan end-2-test menggunakan cypress.

Technology

  • Web Component
  • Javascript
  • Cypress
  • Jest
