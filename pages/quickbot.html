QuickBot

Description

  • Quickbot

Quickbot adalah project open source saya, yaitu sebuah aplikasi automation berbasis console yang dapat mengenerate pembuatan sosial media seperti gmail. Dengan Quickbot, secara otomatis gmail akan dibuatkan sebanyak yang ingin kita tentukan

Project ini dibuat menggunakan spring boot framework java, dan menggunakan selenium untuk proses automation nya. Serta beberapa teknologi pendukung lainnya seperti postgreSQL, redis, kafka hibernate ORM dll.

Technology

  • Java
  • Spring
  • PostgreSQL
  • Kafka
  • ORM Hibernate
  • Selenium
