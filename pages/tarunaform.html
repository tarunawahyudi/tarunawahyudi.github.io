Taruna Form

Description

Taruna Form adalah plugin wordpress yang saya buat untuk client freelance saya. Taruna Form merupkan plugin untuk membuat undangan acara seperti pernikahan dan lain-lain secara instant menggunakan generator. User hanya perlu menginputkan informasi tentang acara nya, maka otomatis plugin akan mengenerate nya.

Plugin ini dibuat mengikuti standar dokumentasi API wordpress developer, menggunakan PHP dan juga jquery. Plugin ini suksess digunakan oleh client hingga saat ini plugin telah membuat banyak undangan acara.

Technology

  • PHP
  • Wordpress
  • jQuery
