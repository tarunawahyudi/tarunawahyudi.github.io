Sistem Informasi Perpustakaan

Description

Sistem Informasi Perpustakaan atau disingkat SIMP adalah proyek freelance saya. SIMP berbasis website, yang mana terdapat akses untuk admin dan juga akses untuk user secara public.

Singkat nya diwebsite ini user dapat melihat daftar buku-buku yang tersedia diperpustakaan dan melihat apakah buku-buku tersebut tersedia untuk pinjam atau sedang dipinjan orang lain. User dapat meminjam sesuai waktu yang ditentukan dengan menunjukan nomer identitas yang sudah terdaftar di database.

Jika ada keterlambatan pengembalian user akan dikenakan denda sebesar yang telah ditentukan oleh admin.

Technology

  • PHP
  • Codeigniter 3
  • MySQL
  • Bootstrap
