Single Sign On

Description

Project ini adalah sebuah project web app yang mengimplementasikan Single Sign On, dimana setiap service yang berada pada cluster yang sama, akan memiliki 1 akses auhtentikasi, yaitu web ini.

Project ini dibuat menggunakan vueJS untuk front end nya dan Laravel Lumen untuk backend nya.

Technology

  • PHP
  • VueJS
  • Typescript
  • Laravel Lumen
  • PostgresSQL
