Bookshelf App

Description

  • Bookshelf App

Saya telah mengembangkan aplikasi Bookshelf menggunakan vanilla JavaScript dan DOM sebagai bagian dari syarat kelulusan di Dicoding Indonesia. Proyek ini dirancang untuk memenuhi tugas submission dan bertujuan memberikan pengalaman pengguna yang sederhana namun efektif dalam mengelola koleksi buku.

Aplikasi Bookshelf memungkinkan pengguna untuk menambah buku baru, memindahkan buku antara kategori seperti "Belum Selesai Dibaca" dan "Selesai Dibaca," serta mencari buku berdasarkan judul. Dengan menggunakan vanilla JavaScript, aplikasi ini memanfaatkan manipulasi DOM secara langsung untuk memastikan kinerja yang optimal dan antarmuka yang responsif.

Proyek ini menekankan penerapan prinsip-prinsip dasar pengembangan web, termasuk pengelolaan state, struktur kode yang rapi, dan penggunaan API untuk menyimpan dan mengambil data buku. Dengan aplikasi ini, saya tidak hanya menyelesaikan syarat untuk lulus, tetapi juga menunjukkan kemampuan saya dalam membangun aplikasi web yang fungsional dan mudah digunakan tanpa bergantung pada framework tambahan.

Technology
