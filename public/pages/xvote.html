XVote

Description

xvote adalah project freelance saya yang digunakan user untuk membuat sebuah sistem pemilihan atau sistem voting elektronik yang menggunakan teknologi blockchain untuk menjaga transaparasi data.

user dapat membuat pemilihan dan menambahkan kandidat-kandidat didalam pemilihan tersebut. Kemudian user lainnya atau bisa disebut sebagai voter, memilki akses untuk melakukan pemilihan diwebsite dan juga melihat hasil data nya secara realtime.

Technology

  • PHP
  • Laravel
  • Tailwind CSS
  • Flowbit
  • Bun Js
  • Blade Component
